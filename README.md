# BYO-Load-Balancer

> (Bring Your Own) Load Balancer

## Developing

- To initialize or clean the build directory

```sh
./scripts/clean.sh
```

- To make a clean build and run the application

```sh
./scripts/run.sh
```

- Run with the config file

```sh
CONFIG_FILE=byolb.config LOG_ERRORS=1 ./scripts/run.sh
```
